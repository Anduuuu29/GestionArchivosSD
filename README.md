## Justicación del Problema
En los lugares donde se trabaja con grandes documentaciones siempre existirá el problema de la organización de esta. Si es en papel, estos se pueden perder o dañar fisicamente, si es digital pueden quedar perdidos o borrados  dado que no siempre hay un orden o una organización para estos. Este problema puede causar retrasos con trabajos o papeleos, debido a que se debe buscar primero el archivo dentro de un mar de documentos, el cual incluso podría no estar. Esto no deberia ocurrir, los archivos deberian tener un orden y un lugar común donde se encuentren todos para que así todo el municipio pueda acceder a estos cuando lo vean conveniente, de una manera rapida y eficaz.
## Requerimientos
### Funcionales
* Capacidad de subir archivos e identificarlos
* Clasificar archivos según parametros dados
* Eliminación de archivos repetidos
* Gestión de archivos mediante filtrado
* Clasificación de procesos (Terminado / No terminado)
* Limpieza de archivos con más de cierta cantidad de tiempo de antiguedad
* 

### No funcionales
* Facil de utilizar
* Cumplimiento de las normal nacionales
* Controles de seguridad en los documentos
