## Justicación del Problema
En los lugares donde se trabaja con grandes documentaciones siempre existirá el problema de la organización de esta. Si es en papel, estos se pueden perder o dañar físicamente, si es digital pueden quedar perdidos o borrados dado que no siempre hay un orden o una organización para estos. Este problema puede causar retrasos con trabajos o papeleos, debido a que se debe buscar primero el archivo dentro de un mar de documentos, el cuál incluso podría no estar. Esto no debería ocurrir, los archivos deberían tener un orden y un lugar común donde se encuentren todos para que así todo el municipio pueda acceder a estos cuando lo vean conveniente, de una manera rapida y eficaz.
## Requerimientos
### Funcionales
* Capacidad de subir archivos e identificarlos. (Admin / User)
* Clasificar archivos según parametros dados. (Admin)
* Eliminación de archivos repetidos. (Admin)
* Busqueda de archivos mediante filtrado. (Admin / User)
* Clasificación de procesos (Terminado / No terminado). (Admin)
* Limpieza de archivos con más de cierta cantidad de tiempo de antigüedad. (Admin)
* Abrir ticket de soporte(User)

### No funcionales
* Fácil de utilizar.
* Cumplimiento del FrameWork del Gobierno.
* Controles de seguridad en los documentos.
* Accesible para todo público (Vecinos y Personas de la Municipalidad)
